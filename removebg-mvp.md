# 图片背景移除工具 - MVP 需求文档

## 1. 项目概述

**项目名称：** RemoveBG Mini
**项目类型：** 轻量级在线图片背景移除工具
**核心功能：** 用户上传图片，自动移除背景并支持下载结果图
**目标用户：** 设计师、电商卖家、自媒体创作者

---

## 2. 技术方案

### 前端
- 纯静态页面（HTML + CSS + JavaScript）
- 拖拽或点击上传图片
- 实时预览处理结果
- 支持下载 PNG 透明背景图

### 后端
- **Runtime：** Cloudflare Workers
- **图片处理：** 调用 Remove.bg API（不自行处理）
- **图片流转：** 全程内存流，不落盘

### API 集成
- Remove.bg API Key（用户自行配置）
- API 端点：`https://api.remove.bg/v1.0/removebg`
- 免费额度：50张/月

---

## 3. 功能列表

### 必须（MVP）

| 功能 | 描述 |
|------|------|
| 图片上传 | 支持拖拽或点击上传，接受 JPG/PNG/WebP，单文件 ≤ 10MB |
| 背景移除 | 调用 Remove.bg API 处理图片 |
| 结果预览 | 处理完成后实时展示原图/结果图对比 |
| 图片下载 | 一键下载结果图（PNG 透明背景） |
| 错误处理 | 文件过大、API 失败、网络异常等场景友好提示 |

### 可选（后续迭代）
- 批量上传
- 自定义 API Key 配置
- 处理记录/历史
- 暗色模式

---

## 4. 用户流程

```
用户访问页面
    ↓
上传图片（拖拽/点击）
    ↓
点击"移除背景"按钮
    ↓
显示 loading 状态
    ↓
调用 Remove.bg API
    ↓
返回结果 → 展示对比图
    ↓
用户点击下载 → 保存结果图
```

## 6. 环境变量

| 变量名 | 说明 |
|--------|------|
| `REMOVE_BG_API_KEY` | Remove.bg API Key |

---

## 7. 非功能性要求

- **性能：** Worker 冷启动 < 200ms，API 响应视 Remove.bg 而定
- **安全：** API Key 存于 Cloudflare 环境变量，不暴露在前端
- **限制：** 单文件 ≤ 10MB，不支持 GIF 动图
